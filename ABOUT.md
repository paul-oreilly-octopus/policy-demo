# Policy Demo

Octopus Deploy demo space showing Platform Hub policies in action. Themed around **Muffin Man Inc**, a fictional global fast-food chain that recently acquired German bakery chain **Crusty Croissant Co**.

Seven pipelines, each blocked by a different governance policy, with live "make compliant → ship" demo flows. Models the same mock-infrastructure approach as `oil-vendor-demo` (Cloud Region targets, runbook-driven mock releases) so the demo is fully self-contained and resettable.

Target audience: platform engineering leaders and technical buyers evaluating Octopus Platform Hub for governance use cases.
